GestureAI is an Android application that performs real-time sign language detection using the device camera and a TensorFlow Lite model.

## Features
- Live camera preview using CameraX
- Real-time sign classification
- On-screen prediction output
- Lightweight TensorFlow Lite model

## Tech Stack
- Kotlin
- Android CameraX
- TensorFlow Lite
- Android Studio

## How to Run
1. Open the project in Android Studio
2. Build and run on a real Android device
3. Grant camera permission

## APK
The compiled APK is included in this repository for quick testing.

## Notes
This project demonstrates the application of computer vision and machine learning for sign language interpretation.
